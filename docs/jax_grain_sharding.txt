grain:
    - sharding splits data across Processes (Pods/Nodes)
    - a.k.a. coarse-grained sharding
jax:
    - via shmap, pmap
    - sharding splits data within 1 process across its
      devices (cores, TPUs, GPUs)

e.g. a cluster of TPUs running 128 processes (shards)
     and each shard having 8 TPU cores.
    - grain distributes 1/128 of data files to each process
    - jax distributes that 1/128 of data across the
      8 local TPU cores.
