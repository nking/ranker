Kind (Kubernetes IN Docker) is a tool for running local Kubernetes 
clusters using Docker containers as "nodes."

Because Kind containers share the Linux kernel with your host 
machine (assuming you are on Linux or using Docker Desktop), 
there is virtually zero virtualization overhead. This makes 
it the fastest way to boot a cluster for development and CI/CD.
``
requires go and docker

for the bash scripts:
   install kubectl (specific to platform OS... I used snap)
for the python SDK:
   pip install kubernetes


Install:
- go install sigs.k8s.io/kind@v0.31.0 (for go 1.17+). 
  This put kind in $(go env GOPATH)/bin.
  which is $HOME/go/bin on my machine, but might be different for others
  - edit ~/.bashrc to include export PATH=$PATH:$HOME/go/bin
    source ~/.bashrc
- install kubectl:
   sudo snap install kubectl --classic

documentation:
  https://kind.sigs.k8s.io/
